About IPL AI

🧠 What is IPL AI?

IPL AI is a machine learning powered cricket analytics platform built on historical IPL data spanning 2008–2025. It combines deep learning models with real-time ball-by-ball tracking to give fans, analysts and fantasy players data-driven insights.

Every prediction is backed by statistical models trained on over 200,000 ball-by-ball records.

🎯 Features

  • Match winner prediction with confidence score
  • Live score forecasting using LSTM neural network
  • Head-to-head rivalry breakdown with venue stats
  • Ball-by-ball live innings tracker
  • Dream XI builder with fantasy scoring
  • Player vs player stat comparison
  • Season-wise top performers per team
  • All-time run scorers and wicket takers

💾 ML Models

  • Match Predictor — LSTM trained on team, toss, venue features
  • Score Predictor — LSTM sequence model on mid-innings state
  • Dream XI — Fantasy scoring algorithm with role classification
  • Player Search — Two-stage prefix + fuzzy index (sub-100ms)
  • Models retrained every season with new IPL data

🛠️ Tech Stack

  • Backend — Python · Flask · TensorFlow / Keras
  • ML — scikit-learn · NumPy · Pandas
  • Search — RapidFuzz for fuzzy player matching
  • Data — IPL ball-by-ball dataset 2008–2025
  • Frontend — Vanilla JS · HTML5 Canvas · CSS3 animations

📈 Data Coverage

  • All IPL seasons from 2008 to 2025
  • Every ball bowled in every match
  • 33 venues across India, UAE & South Africa
  • 10 current franchises + 6 defunct teams
  • 500+ unique players tracked across all seasons

⚡ Performance

  • Player search returns results in <100ms using prefix index
  • All team name variants normalized at startup
  • Venue names consolidated to 33 canonical entries
  • Models loaded once at startup — zero reload per request
  • Cricket field background drawn on Canvas — GPU accelerated
